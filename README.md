# Notification System Project

This repository contains the implementation of a cross-platform notification system. The project is divided into four main tasks, each residing in its own directory.

## Project Structure

### [Task 1: Database Design](task1-database/database_design.md)
Contains the database schema design and documentation for the notification system.
- **Location**: `task1-database/`
- **Key File**: [`database_design.md`](task1-database/database_design.md)

### [Task 2: System Architecture](task2-architecture/architecture_design.md)
Details the high-level system architecture, including component interactions and data flow.
- **Location**: `task2-architecture/`
- **Key File**: [`architecture_design.md`](task2-architecture/architecture_design.md)

### [Task 3: Mobile Application](task3-mobile-app/README.md)
The React Native mobile application source code.
- **Location**: `task3-mobile-app/`
- **Key File**: [`README.md`](task3-mobile-app/README.md) (Contains local setup and run instructions)

### [Task 4: Admin Panel](task4-admin-panel/README.md)
The admin panel for managing notifications and users.
- **Location**: `task4-admin-panel/`
- **Key File**: [`README.md`](task4-admin-panel/README.md)
